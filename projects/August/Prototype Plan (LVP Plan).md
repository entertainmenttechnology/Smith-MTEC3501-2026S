<H1>Least Viable Product (LVP) – Target for Culmination (Next Semester)</H1>

<H3>What will the user experience?</H3>
The user will be able to experince the enigma machine working. I plan on creating an enigma machine with 3 rotors that can function. 
The enigma machine will be set up as part of an exibit explaing it's history as well as how it functions. 

<H3>What features must be complete for it to be “done”?</H3>
Three rotors, a keyboard, a light display, the internal mechanics, and a small exibit with placards.

<H3>What technologies or platforms are involved?</H3>
CAD, 3d printing, PCB design and creation, Laser cutting.

<H3>How will you measure success?</H3>
Success will mostly be mesasured in how other people interact with the exibit as well as how functional the device is.
