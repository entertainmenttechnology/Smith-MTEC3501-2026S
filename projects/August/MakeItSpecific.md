1. Working Title
The Enigma Machine Recreation Project 

2. Project Description 
A recreation of the Enigma Machine used during WWII using modern manufacturing techniques.

4. Project Format
Computer Numerical Control and Additive manufacturing in conjunction with Computer Aided Design

6. User / Integrator Experience
The user of this modernized enigma machine will be able to send and receive messages that they encode and decode using analog mechanisms.
The intended experience is focused around the idea of modern vs early forms of encryption and the idea that every system is breakable. 
Eventually this system could be shown in a museum or similar location focusing on how data is encrypted/decrypted. With an enigma machine compared to other methods and how the same methods can be cracked.
If you are building assets, the “user” is the developer, designer, performer, or system builder who integrates them.

5. System Description
The main project will be built using Autodesk inventor for CAD, Git for version control, prusa slicer for file preparation and a prusa mk4s both with and without a mmu3. If larger scale electronic systems are used, ki-cad and JLCPCB would be used for circuit board design and manufacturing. Parts that aren't able to me manufactured in house will be purchased from Mcmaster carr or similar vendors.

7. Explicit Dependency
This project depends on my cad and other design abilities as well as the mechanical understanding of the underlying parts of the enigma machine.
