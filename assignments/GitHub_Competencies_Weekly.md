## Week 1: GitHub Foundations
**Setup & Orientation**
- Create GitHub account
- Install GitHub Desktop
- Install Visual Studio Code
- Access course repository
- Navigate repository structure in browser
- Understand repository folders and organization

**Deliverables:**
- GitHub account created and verified
- GitHub Desktop installed and authenticated
- VS Code installed
- Repository cloned locally

## Week 2: Basic Repository Operations
**Skills:**
- Branch from main
- Create project folder structure
- Create and edit README.md files
- Upload documentation to repository
- Basic Markdown formatting

**Deliverables:**
- Personal project folder created
- Branch created from main
- README.md created in project folder
- Brainstorming documentation uploaded

**Deliverables:**

## Week 7: Feedback & Comments
**Skills:**
- Use issue comments for feedback
- Create feedback templates
- Reference other issues (@mentions)

## Week 8: Branching & Merging

- Prototype branch created

- Branch merged when appropriate

## Summary of GitHub Skills Introduced

### Foundational (Weeks 1-3)
- Account setup and repository access
- Branching and basic commits
- Markdown formatting
- Issues and labels

### Intermediate (Weeks 4-7)
- Project Boards
- Milestones
- Asset management
- Feedback workflows

### Advanced (Weeks 8-11)
- Branching strategies
- Pull Requests and code review
- Workflow tracking
- Version control best practices

### Professional (Weeks 12-15)
- Repository organization
- Documentation standards
- Freeze and release procedures
- Archival and reflection

---

*This progression ensures students build GitHub competency gradually, with each skill building on the previous week's foundation.*
